# Morse-Code-Translator

## Outline

The project Morse Code Translator is about creating a web page that translates English->Morse and Morse->English.
The project that uses objects, complex data, and DOM manipulation in JavaScript and HTML to allow interactive translation.

## MVP

- A user interface that allows the user to either input some English text or some Morse Code.
- The user has option to select and JS functions are created that would allow the user to:

  - translate their English text(A-Z) into Morse Code
  - Morse Code into English text
